## Installation

```bash
1. clone the directory.
2. Use the Linux package manager apt, yum, dnf to install sshpass package.
3. chmod +x ssh.sh
4. python3 run.py
```
> [!WARNING]
> Do not rename ssh.sh file if you do then you need to edit the name inside run.py.
